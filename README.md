# webind
Required Packages:
  python3-pip,
  python3-dev, 
  default-libmysqlclient-dev, 
  build-essential, 
  libpq-dev.
  
Required pip modules:
	wheel,
	flask,
	flask-mysqldb,
	flask-sqlalchemy,
	pyyaml,
	paramiko,
	psycopg2 (driver for PostgreSQL),
	pymysql (driver for MySQL).
	
after download run into webind directory: python3 -m venv flask --upgrade
